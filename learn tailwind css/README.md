# Learn Tailwind CSS

A small starter folder for practicing Tailwind CSS utilities.

## Files

- `index.html` for quick CDN-based preview
- `src/input.css` for the CLI build source
- `tailwind.config.js` for Tailwind configuration
- `package.json` for the Tailwind CLI scripts

## Next step

Run `npm install` inside this folder if you want to use the build workflow.
